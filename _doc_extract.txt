2.4 Motion & Animation Rules
Page load stagger
Hero elements enter with staggered fade+slide-up: badge → H1 → subtext → trust chips → buttons. Each 150ms apart.
Scroll reveal
IntersectionObserver at 15% threshold. Elements fade+slide-up (y:24px → 0, opacity:0→1). 400ms ease-out.
Stats count-up
requestAnimationFrame counter from 0 to target over 1800ms. Easing: cubic. Triggers once on viewport entry.
Card hover
translateY(-5px), shadow deepen, left border 3px green appears. 220ms cubic-bezier(0.4,0,0.2,1).
Tab transitions
Content fades out (100ms), new content fades in (200ms). No layout shift.
Timeline steps
Sequential reveal on scroll — each step fades in 200ms after the previous.
Particle / ambient
Subtle floating dots or grid pattern in hero — CSS only, no heavy canvas.
DO NOT USE
Parallax scroll, auto-carousels without controls, flicker/flash effects, heavy canvas animations.
